# 14. Notes App

This project is a **Notes App**.  
It helps you write and organize text notes.  

---

## Features

- Input: Note title and content.  
- Output: List of saved notes.  
- Supports basic operations:  
  - Add a note  
  - Edit a note  
  - Delete a note  
  - List all notes  
- Can be extended with tags or search functionality.
